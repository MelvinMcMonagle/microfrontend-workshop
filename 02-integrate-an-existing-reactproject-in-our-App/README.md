# Building Micro Frontends with React, Vue, and Single-spa 

## Getting started


2. Install dependencies

```
yarn

# or

npm install
```

3. Start the app

```sh
npm start
```

4. Visit the following URLS:

```sh
# renders both apps
http://localhost:8080/

# renders only react
http://localhost:8080/react

# renders only vue
http://localhost:8080/vue
```
